## ğŸ“Œ Project Overview
This project presents a comprehensive analysis of **bank transaction data** designed to reflect real-world financial operations.  
The dataset contains **9,999 transactions** recorded across multiple branches in Indonesia, covering diverse channels (ATM, Mobile Apps, Teller, Internet Banking), customer segments (Premium, Regular, Corporate), transaction types (Transfer, Payment, Deposit, Withdrawal), and currencies.  

The dashboard was developed in **Power BI**, inspired by the professional layout of a Bank Loan Report, and adapted to highlight transaction performance.  
It provides a clear view of key performance indicators (KPIs), transaction status monitoring, customer distribution, and time-based trends, enabling stakeholders to evaluate operational efficiency and customer behavior in the banking domain.

## ğŸ¯ Objectives
- Provide a clear overview of **bank transaction performance** across branches, channels, and customer segments.
